# 角色书

`角色书` 是一款面向单机使用的角色设定与剧情创作工具，支持在手机上离线整理人物资料、记录剧情灵感，并以本地方式保存数据。

## 软件功能

- 角色管理
  - 新建、编辑、删除角色
  - 设置角色姓名、性别、发型、瞳色、身高等基础信息
  - 补充更多自定义设定
  - 记录人物经历

- 剧情管理
  - 新建剧情或灵感记录
  - 支持搜索、查看、编辑、删除
  - 支持批量操作
  - 剧情内容以卡片方式组织，便于分段整理

- 作品管理
  - 为每个作品单独管理角色和剧情
  - 首页可快速进入对应作品
  - 作品信息会随内容更新而同步维护

- 本地保存
  - 数据保存在用户设备本地
  - 不依赖网络即可使用
  - 适合离线写作、随手记录和长期整理

- Android 安装包
  - 可打包为安卓 APK
  - 安装后可直接作为单机 App 使用
  - 适合在手机上随时查看和编辑

## 主要界面

- 首页：查看作品列表，快速进入角色页或剧情页
- 人设页：编辑角色卡与人物设定
- 剧情列表：浏览、搜索和管理剧情内容
- 剧情编辑页：按卡片拆分剧情结构，方便拆解和重组内容

## 使用方式

### 本地运行

```bash
npm install
npm run dev
```

然后在浏览器中打开 Vite 提供的本地地址即可预览。

### 打包 Android APK

```bash
npm run build
npx cap copy android
cd android
gradlew assembleDebug
```

生成的测试包通常位于：

```text
android/app/build/outputs/apk/debug/app-debug.apk
```

## 适用场景

- 角色设定整理
- 剧情大纲记录
- 单机写作辅助
- 灵感碎片收集
- 手机端离线创作

## 说明

- 当前项目以本地使用为主，不依赖在线服务。
- 如果需要长期维护或扩展，也可以继续加入更完整的本地数据库、导出能力或发布版签名流程。
